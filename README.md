## 시작가이드
### Repository clone
~~~
git clone https://github.com/MyStraw/miniproject
~~~
### DB 구성
- 레파지토리 내 DB 폴더의 최신 날짜 폴더안의 .sql 파일 MySQL에서 불러오기후 실행
- uploads 폴더를 c:\Temp\uploads 경로에 배치  


### 백엔드
- Metro 프로젝트 서버 실행

## 주요 기능
- 부산 지하철 운행정보, 역 정보, 편의시설 정보  
- 회원가입 및 로그인  
- 게시판
- 이미지 업로드  
- 좋아요  
- 내가 쓴글 및 내가 좋아요 눌린 글 모아보기
